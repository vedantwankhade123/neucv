# NEUCV - Resume Builder

NEUCV is a modern, intuitive resume builder that helps you create professional, standout resumes in minutes. With a variety of templates, easy customization options, and a real-time preview, crafting the perfect resume has never been easier.

This project was built with the assistance of **Dyad**, an AI coding partner that helps create and modify web applications in real-time.
